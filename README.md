# Geospatial Data Analysis

## Overview
This project focuses on analyzing location-based business data to identify optimal areas for business expansion. The analysis helps visualize sales geographically, compare regional performance, and detect areas with high demand but low business presence.

## Objective
The main objective of this project is to analyze geospatial sales data, identify high-demand regions, and provide insights that help businesses choose better expansion locations.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Folium
- Google Colab

## Steps Performed
- Loaded the geospatial dataset
- Inspected rows, columns, missing values, and duplicates
- Removed duplicate rows
- Cleaned and standardized column names
- Handled missing values
- Converted sales, latitude, and longitude columns to proper numeric format
- Analyzed sales by city/region
- Identified top-performing locations by sales
- Created a geographic marker map for location-based sales
- Generated a heatmap to visualize demand concentration
- Highlighted potential expansion areas based on high demand and low presence
- Saved the cleaned dataset for further analysis

## Files in this Repository
- `Geospatial_Data_Analysis.ipynb` → Colab notebook containing the full analysis
- `cleaned_geospatial_data.csv` → Cleaned geospatial dataset
- `README.md` → Project documentation

## Key Insights
The analysis helps identify the regions with strong sales concentration and potential demand. These insights can help businesses locate underserved areas and make smarter expansion decisions.

## Outcome
The dataset was successfully cleaned, analyzed, and visualized to understand geographic demand patterns and identify business expansion opportunities.
